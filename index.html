Explore Mandelbrot and Julia Fractals in Real-Time

Synaptory Fractal Traveler is a GPU-accelerated web application for exploring the Mandelbrot set and Julia sets in real-time. Built with WebGL, it renders fractals directly on your GPU, allowing smooth zooming, panning, and rotation through the infinite complexity of fractal geometry.

The Mandelbrot set is one of the most famous objects in mathematics, revealing infinitely detailed structures at every scale. As you zoom deeper, you discover spirals, seahorse valleys, and intricate self-similar patterns that emerge from a simple equation: z = z² + c. Each point on the Mandelbrot set boundary corresponds to a unique Julia set, and this application lets you explore both in real-time.

Navigate using mouse controls: scroll to zoom, drag to pan, double-click to center and zoom, or right-drag to rotate. Touch controls are fully supported: pinch to zoom, drag to pan, and use two fingers to rotate. Choose from over 30 curated views showcasing fascinating locations like Seahorse Valley, Misiurewicz Points, and the Corona region. Switch between color palettes including Event Horizon, Aurora, Fire, and Ocean to transform the visual experience.

Julia set dives animate through parameter space, creating mesmerizing real-time transitions between fractal forms. Save your favorite views locally, capture high-quality screenshots, and share exact coordinates via URL. The adaptive quality system automatically adjusts rendering precision based on your GPU performance.

Fractal Traveler is free and open-source under the MIT license. Visit the fractal gallery for curated screenshots, read the documentation, or explore the source code on GitHub.
